    Récupération

    Si vous possédez Git, tapez dans un invité de commande:

    $ cd path/to/www
$ git clone https://github.com/guillaume-sainthillier/iscore.git

    Si vous ne possédez pas Git, téléchargez le projet au format Zip ou Tar.gz, et extrayez le dans le dossier www de votre serveur apache.

    Déploiement

    Adaptez la variable $infos dans le fichier fichier dyn/database.inc.php suivant la configuration de votre serveur.
    Importez le fichier iscore_structure.sql dans votre base de données.
    Si besoin, importez le fichier iscore_donnees.sql pour un petit jeu de données.

    Pré requis

    Vous devez avoir installé:

    • un serveur apache et PHP5.3+
    • une base de données MySQL

    En fait, reportez vous aux installations de :

    • sous Windows : WAMP ou easyphp
    • sous Linux : LAMP

    Si vous débutez, il est plus simple de garder MySQL qui est packagé avec wamp ou easyphp.
